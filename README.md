# 説明
RUBY合宿2021春にて、DXrubyを用いたスクロールアクションを取り入れた音楽ゲームです。

## （ゲーム画面）
・操作キャラクターは上攻撃と下攻撃ができます。対応するキーはそれぞれQキーとAキーです。
・画面右側から流れてくる音符と判定位置の距離によって、得点を獲得できます。
音符と判定位置が近い順に、
Great:1000点
Good:500点
Bad:100点
となります。
